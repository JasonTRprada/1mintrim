이미지를 아이콘(.ico)으로 변환

업로드도, 서버 전송도 없습니다. 모든 처리는 브라우저 안에서만 일어납니다.

이미지를 여기에 드래그하거나 클릭해서 선택하세요

최대 5MB까지 지원합니다 (PNG, JPG 등)

정사각형 영역을 드래그해서 옮기고, 모서리를 드래그해서 크기를 조절하세요. 이 영역만 아이콘으로 변환됩니다.

생성할 아이콘에 다음 크기들이 모두 포함됩니다: 16×16, 32×32, 48×48, 64×64, 128×128, 256×256

사용법

  1. 이미지 업로드 — 정사각형에 가까운 이미지일수록 결과물이 깨지지 않고 예쁘게 나옵니다.
  2. 변환 — "ICO로 변환" 버튼을 누르면 여러 크기가 포함된 .ico 파일이 즉시 다운로드됩니다.

자주 묻는 질문

ICO 파일은 어디에 쓰나요?

윈도우 프로그램 아이콘, 웹사이트 파비콘(favicon.ico) 등에 사용됩니다.

왜 여러 크기가 한 파일에 들어가나요?

ICO 파일 형식은 여러 해상도를 하나의 파일에 담을 수 있어, 운영체제가 상황에 맞는 크기를 자동으로 골라 씁니다.

이미지가 서버에 업로드되나요?

아니요. 모든 변환은 브라우저의 Canvas API로 처리되며 서버로 전송되지 않습니다.
