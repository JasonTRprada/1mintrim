영상 구간을 GIF로 변환

최대 10초 구간을 골라 GIF로 만듭니다. 모든 처리는 브라우저 안에서만 일어납니다.

영상을 여기에 드래그하거나 클릭해서 선택하세요

최대 1분 영상, 최대 30MB까지 지원합니다

0.0s 구간: 0.0s (최대 10.0s) 0.0s

사용법

  1. 영상 업로드 — 최대 1분, 30MB 이하 영상만 가능합니다.
  2. 구간 선택 — 타임라인에서 GIF로 만들 구간을 고르세요. 최대 10초까지 선택할 수 있습니다.
  3. 재생 방향 선택 — 일반 순방향(A → B) 또는 갔다가 되돌아오는 부메랑(A ⇄ B) 중 선택하세요.
  4. 변환 — "GIF로 변환" 버튼을 누르면 브라우저 안에서 GIF 파일이 만들어져 바로 다운로드됩니다.

자주 묻는 질문

왜 10초로 제한하나요?

GIF는 영상보다 압축 효율이 낮아서 길이가 늘어날수록 파일 용량이 급격히 커집니다. 공유하기 좋은 크기를 유지하기 위해 10초로 제한했습니다.

부메랑(A ⇄ B)이 뭔가요?

선택한 구간을 정방향으로 재생한 뒤 역방향으로 한 번 더 재생해서 자연스럽게 왔다 갔다 하는 루프 효과를 만드는 기능입니다.

영상이 서버에 업로드되나요?

아니요. 모든 변환 작업은 브라우저 안에서 ffmpeg.wasm을 통해 처리됩니다.
