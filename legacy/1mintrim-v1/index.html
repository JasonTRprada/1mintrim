    사용법

    1. 영상 업로드 — 위 영역에 영상 파일을 드래그하거나 클릭해서 선택하세요. 300MB 이하 영상을 지원합니다.
    2. 구간 선택 — 타임라인 아래의 두 핸들을 드래그해서 시작점과 끝점을 지정하세요. 미리보기 영상으로 결과를 바로 확인할 수 있습니다.
    3. 다운로드 — "트림해서 다운로드" 버튼을 누르면 브라우저 안에서 잘라낸 구간만 새 파일로 만들어 즉시 다운로드됩니다.

    왜 만들었나요

    짧은 영상 클립 하나를 자르려고 무거운 편집 프로그램을 설치하거나, 회원가입까지 해야 하는 웹 서비스에 영상을 업로드하는 게 번거로워서 만들었습니다. 1MinTrim은 딱 한 가지 일만 합니다 — 1분 이하 영상을 원하는 구간만 잘라내는 것. 그 외 기능은 일부러 넣지 않았습니다.

    모든 처리는 브라우저 내부에서 WebAssembly로 동작하는 ffmpeg를 이용해 이루어집니다. 영상 파일이 외부 서버로 전송되지 않으며, 페이지를 닫으면 아무것도 남지 않습니다.

    자주 묻는 질문

    영상이 서버에 업로드되나요?

    아니요. 모든 트림 작업은 사용자의 브라우저 안에서 ffmpeg.wasm을 통해 처리됩니다. 영상 파일은 한 번도 외부로 전송되지 않습니다.

    소스 영상 길이 제한이 있나요?

    소스 영상 길이 제한은 없습니다. 300MB 이하라면 긴 영상에서도 원하는 구간을 잘라낼 수 있습니다. 단, 구간 하나는 최대 1분, 이어붙인 총 출력은 최대 5분입니다.

    화질 손상 없이 자를 수 있나요?

    가능하면 재인코딩 없이 키프레임 단위로 무손실 트림을 시도합니다. 다만 선택한 구간이 키프레임과 정확히 맞지 않으면 정밀도를 위해 재인코딩 방식으로 자동 전환될 수 있습니다.

    어떤 영상 형식을 지원하나요?

    대부분의 브라우저가 재생할 수 있는 형식(MP4, WebM, MOV 등)을 지원합니다. 출력은 MP4로 제공됩니다.

    여러 영상을 합치거나 효과를 넣을 수 있나요?

    아니요. 1MinTrim은 트림(자르기) 기능 하나에만 집중합니다. 다른 편집 기능은 지원하지 않습니다.

    비용이 드나요?

    아니요. 완전히 무료이며 가입 절차도 없습니다.
